## 🖼️ 제작 방법 예시 (가장 간단)

### **Photoshop**
```
1. 파일 → 새로 만들기
2. 크기: 200×200px (위 표에서 "제작 크기" 선택)
3. 배경: 투명으로
4. 그림 그리기
5. 파일 → 내보내기 → PNG
   ⚠️ "저장 크기"로 리사이징 후 저장
```

### **Figma / Adobe XD**
```
1. 아트보드 생성: 200×200px
2. 디자인 완성
3. 내보내기 → PNG
   ⚠️ 스케일 100% 또는 2배 선택
```

### **온라인 (Pixlr.com, Canva)**
```
1. 새 프로젝트: 200×200px
2. 디자인 완성
3. Download PNG (200×200px 다운로드됨)
4. (선택) 온라인 리사이저로 "저장 크기"로 축소
   예: 200×200 → 100×100
```

### **간단한 온라인 리사이저**
- https://pixlr.com (Pixlr)
- https://online-convert.com
- https://www.img2go.com

## 📌 핵심 체크리스트

- [ ] 각 이미지를 위 표의 "저장 크기"로 준비
- [ ] 모두 PNG 포맷 (투명 배경)
- [ ] 폴더: `assets/` 아래에 정렬
- [ ] 파일명: 소문자 + 하이픈 (예: `icon-gel.png`)
- [ ] HTML `window.GAME_ASSETS`에 경로 입력
- [ ] 로컬 서버로 테스트
