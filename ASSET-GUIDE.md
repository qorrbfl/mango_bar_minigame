# 🎨 PNG 에셋 교체 가이드

두 게임 모두 **에셋 매니페스트 방식**으로 리팩터링되어 있습니다.
각 HTML 파일 상단의 `window.GAME_ASSETS = { ... }` 설정에서 `null` 자리에 이미지 경로를 넣으면
해당 부위가 PNG로 교체되고, `null`로 두면 기본 SVG 그림이 그대로 사용됩니다.
**부분 교체 가능** — 원하는 것부터 하나씩 바꿔가며 확인하세요.

## 경로 넣는 3가지 방법

```js
window.GAME_ASSETS = {
  // 1) 같은 폴더의 상대 경로 (assets 폴더를 만들어 넣는 방식 추천)
  tool_brush: 'assets/brush.png',

  // 2) 웹에 올린 이미지 URL
  girl: 'https://내도메인.com/img/girl.png',

  // 3) data URI로 HTML에 직접 내장 (파일 1개 유지 가능, 용량 커짐)
  sparkle: 'data:image/png;base64,iVBORw0KG...',
};
```

> ⚠️ 상대 경로 사용 시: 브라우저 보안 정책 때문에 `file://`로 직접 열면 이미지가 안 뜰 수 있어요.
> 게임 폴더에서 간단 서버를 켜고 확인하세요: `python3 -m http.server` → `http://localhost:8000/spa-girl-game.html`

## 공통 제작 규칙

- **투명 배경 PNG** (배경 제외)
- 권장 크기는 아래 표의 2배(@2x)로 제작하면 레티나에서도 선명 (예: 표에 110×110 → 220×220px 제작)
- "기준점(anchor)"은 이미지가 게임 좌표에 놓일 때의 정렬 기준입니다. 특히 **도구류는 도구 끝(팁)이 기준점**에 오도록 그려야 드래그 조작감이 맞습니다
- 액체가 담기는 용기(믹서기·냄비·몰드)는 **기본 SVG와 비슷한 실루엣**으로 그려야 내용물(퓨레 게이지)이 어긋나지 않아요. 액체·게이지 자체는 계속 코드로 그려집니다

## 🧖 스파걸 게임 (spa-girl-game.html)

게임 좌표계: viewBox 400×440. 얼굴 영역: x 98~302, y 118~380.

### 👩 얼굴 영역 지정 (girl PNG 사용 시 필수!)

팩 바르기·문지르기 판정, 팩/방울이 얼굴 밖으로 안 나가게 하는 클리핑, 오버레이 배치가 전부 **얼굴 타원** 하나를 기준으로 동작합니다. 캐릭터 PNG의 얼굴 위치가 기본 그림과 다르면 파일 상단에서 지정하세요:

```js
window.GAME_FACE = { cx:200, cy:250, rx:100, ry:126, debug:true };
```

- `cx, cy` = 얼굴 타원의 중심, `rx, ry` = 가로/세로 반지름 (게임 좌표, 가로 400 기준)
- **`debug:true`를 켜면 분홍 점선 타원이 화면에 표시**돼서 girl PNG 위에 겹쳐 보며 숫자를 맞출 수 있어요. 맞춘 뒤 `debug`는 지우세요
- `null`로 두면 기본 캐릭터의 얼굴 영역 사용

**PNG 픽셀 → 게임 좌표 환산법**: girl PNG는 게임 좌표 (50,40) 위치에 300×420 크기로 놓입니다. 원본 PNG가 W×H 픽셀이면:

```
게임X = 50 + (PNG의 x픽셀) × 300 / W
게임Y = 40 + (PNG의 y픽셀) × 420 / H
```

예) 600×840px PNG에서 얼굴 중심이 (300, 400)px, 얼굴 폭 380px·높이 480px라면 → `cx = 50+300×0.5 = 200`, `cy = 40+400×0.5 = 240`, `rx = 380×0.5/2 = 95`, `ry = 480×0.5/2 = 120`

타원이 아닌 정밀한 얼굴 윤곽이 필요하면 `clipPath:'M ... Z'`에 SVG 경로를 직접 넣을 수 있어요 (cx/cy/rx/ry는 판정용으로 계속 사용, clipPath는 시각 클리핑 전용).

### 코드로 계속 그려지는 것(교체 불필요)
- 액체/게이지류: 믹서기·냄비·몰드 속 **퓨레 차오름**, 온도 게이지, 진행 바
- 믹서기 안에 쌓이는 망고 조각·설탕 알갱이 (덩어리→퓨레 전환 연출)
- 붓는 줄기(스트림), 말풍선/카운트다운 UI

## 작업 순서 추천

1. `assets/` 폴더를 만들고 PNG를 위 표의 파일 크기·기준점대로 제작
2. HTML 상단 `window.GAME_ASSETS`에서 만든 것부터 경로 입력
3. 로컬 서버로 열어 확인 (`python3 -m http.server`)
4. 위치가 어긋나면: 도구류는 기준점(팁 위치)을, 용기류는 실루엣을 기본 SVG에 맞춰 조정
5. 전부 확정되면 필요 시 data URI로 인라인해서 단일 파일로 배포 가능
